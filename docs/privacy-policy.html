Better History Privacy Policy

Last updated: April 22, 2026

Better History is a Chrome extension that organizes a user’s browsing history into search, sessions, and analytics views inside the new tab page.

Data We Collect

  • Browsing-history metadata such as page titles, URLs, domains, and visit timestamps.
  • Session and engagement data such as visit duration and scroll depth.
  • Extracted page text content from visited web pages to support local full-text search.
  • User preferences such as theme, excluded domains, and storage settings.

How We Use Data

  • To import and display browsing history inside Better History.
  • To enable full-text search over visited pages.
  • To group visits into browsing sessions and show analytics.
  • To apply user settings such as excluded domains and automatic text pruning.

How Data Is Stored

All collected data is stored locally on the user’s device using Chrome extension storage and IndexedDB. Better History does not transmit browsing data to external servers.

Sharing

Better History does not sell, transfer, or share browsing data with third parties.

User Controls

  • Users can export all stored Better History data as JSON.
  • Users can delete all stored Better History data from Settings.
  • Users can exclude specific domains from future tracking.
  • Users can automatically prune older extracted page text.

Contact

For privacy questions, use the support or contact address listed on the Chrome Web Store item for Better History.
